Embarking on a collaborative journey in a school exercise, my team and I faced the challenge of creating a localization manager using WPF in C#.
This project required us to develop a tool that simplifies the management of multilingual applications, catering to both developers and content managers.

My contribution focused on designing and implementing the export functionality, which bridges the gap between data input and its practical use in real-world applications.

Project Overview

The localization manager serves as a versatile tool that allows users to:

Add language keys and values to create a dynamic spreadsheet-like interface.
Import data from CSV, JSON, and XML formats for seamless integration.
Export data in the same formats (CSV, JSON, XML) for maximum compatibility.
Generate usable classes in C# and C++ for straightforward integration into multilingual applications.

Impact and Learnings

This project was not only a technical challenge but also a collaborative endeavor that strengthened our problem-solving and teamwork skills.
